# fifa-u17-data-analysis
This project explores FIFA U17 match events or stats for the 2024-2025 season. I cleaned and combined the dataset to analyze the player events and team statistics.

## Project Highlights 
Identified players who recieved yellow cards, red cards, and scored  goals.
Calculated totals per team using  SQL aggregation (COUNT with NULLIF).
Built an Excel dashboard to visualize team performance and match event trends

## Tool Used 
SQL | Excel | Data Cleaning | Data Visualization

## Insights
Visualized totals for Yellow Cards, Red Cards and Goals per team.
Highlighted team perfomance patterns across the 2024-2025 season
